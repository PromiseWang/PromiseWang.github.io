搭建Hexo博客

基于Windows下安装

1. 下载必要的东西

  • git，安装的时候也没有进行过多的选择，一路的默认
  • node.js
  • 因为npm镜像源很慢，所以使用国内其他的npm，npm install -g cnpm --registry=https://registry.npm.taobao.org

2. 安装Hexo

  • cnpm install -g hexo-cli

安装完毕后可以使用 hexo -v 进行验证是否安装成功。

3. 新建用于存放blog的文件夹

这个文件夹就是关于这个博客的所有东西

4. 生成博客

  • hexo init : 初始化hexo。

5. 博客操作

  • hexo s: hexo server 启动本地服务运行博客

  • hexo n "文件名": hexo new 新建一篇文章(Markdown)

  • hexo clean: 清理

  • hexo g: hexo generating 生成

6. 博客部署至GitHub

  • 新建仓库: PromiseWang.github.io(后续直接可在浏览器直接访问)

  • cnpm install --save hexo-deployer-git: 部署插件

  • 设置_config.yml:

  • hexo d: 部署远端

    (记得输入GitHub的账号和密码)

7. 更换第一个主题

  • https://github.com/litten/hexo-theme-yilia: 使用了这个主题

  • git clone 主题地址 下载位置: git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

  • 配置: 修改_config.yml文件

  • 重新再清理、生成、启动、再上传一下
